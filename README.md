# DJZS ✦ On-Chain Newsletter

A Web3-native newsletter platform with NFT-gated content on Base blockchain. Subscribe by minting an NFT and unlock premium alpha, AI-powered insights, and exclusive content.

## 🌟 Features

- **NFT-Gated Access**: Mint a Subscribe NFT on Base to unlock premium content
- **Web3 Integration**: RainbowKit wallet connection with Base & Base Sepolia support
- **Real-time Subscription Checking**: Automatically verify NFT ownership for content access
- **Premium Content**: Members-only newsletter issues, trade setups, and alpha drops
- **Transferable Subscriptions**: ERC-721 NFTs can be traded or gifted
- **Beautiful UI**: DJZS-branded design with starfield backgrounds and neon effects
