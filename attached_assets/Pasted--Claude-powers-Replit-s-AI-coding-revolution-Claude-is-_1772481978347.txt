# Claude powers Replit's AI coding revolution

**Claude is the core AI engine behind Replit Agent**, Replit's flagship coding assistant that turns natural language into working applications. This is not a plugin or optional add-on — Anthropic's Claude models, accessed via Google Cloud's Vertex AI, serve as the primary intelligence driving Replit's most important product. Since launching Agent in September 2024, Replit's revenue has grown more than **10x** (from ~$2.8M to $150M+ ARR), and the company raised $250M at a $3 billion valuation. Beyond the built-in Agent, the integration spans five additional layers: Claude Code can run directly in Replit's shell, the Claude Agent SDK has official Replit tutorials, a community MCP server connects Claude Desktop to Replit workspaces, Replit offers managed Claude API access for user-built apps, and the two companies co-sponsor hackathons and developer events.

## Replit Agent runs on Claude by default

Replit Agent — the product responsible for nearly all of Replit's explosive growth — uses Claude as its primary language model. Users describe what they want in plain English, and Claude handles the code generation, multi-file editing, debugging, and architectural decisions. Replit wraps Claude in its own agent loop that manages environment setup, database provisioning, and one-click deployment to Google Cloud Run.

Michele Catasta, Replit's President, has been explicit: *"The vast majority of what the agent does is code-based, and Claude is by far the best model available."* Users don't need to configure anything — Claude is built into the subscription (**$20/month** Core or **$100/month** Pro). No API keys required.

The model versions have tracked Claude's evolution closely. Claude 3.5 Sonnet powered Agent v1 (September 2024). Agent v2 launched alongside Claude 3.7 Sonnet in February 2025. Claude Sonnet 4.0 arrived mid-2025, and as of late 2025, **Claude Sonnet 4.5 is the default model** powering Agent 3. Replit reported that Sonnet 4.5 reduced their code editing error rate from **9% to 0%** on internal benchmarks. Agent 3 can work autonomously for up to 200 minutes, self-test applications in a browser, generate bug reports, and fix issues — all built on Claude's reasoning capabilities.

Replit also uses Gemini 1.5 Flash for speed-optimized auxiliary features (code completion, Design Mode), making it a multi-model platform, but Claude handles the heavy lifting.

## Five ways Claude connects to Replit beyond Agent

The built-in Agent is the highest-profile integration, but developers interact with Claude through Replit in several other ways.

**Claude Code in Replit's shell.** Anthropic's terminal-native coding tool can be installed directly inside Replit's workspace by running `npm install -g @anthropic-ai/claude-code` in the shell. This gives developers Claude Code's powerful reasoning and code editing within Replit's cloud infrastructure. It requires the user's own Anthropic API key (stored in Replit Secrets) and operates independently from Replit Agent — the two don't share context. Replit maintains a dedicated landing page at `replit.com/claude-code` promoting this combination. Some developers prefer this approach for cost control, bypassing Agent's credit-based billing.

**Claude Agent SDK.** Replit provides an official tutorial at `docs.replit.com/tutorials/claude-agent-sdk` for building autonomous AI agents using Anthropic's Agent SDK ("Claude Code as a library"). A pre-built template includes scaffolding for agents, tools, MCP servers, and skills. Developers can build multi-agent architectures that connect to external services (Todoist, Slack, Linear, GitHub) and deploy them as scheduled tasks on Replit's infrastructure.

**MCP (Model Context Protocol).** Replit was named as an early adopter in Anthropic's November 2024 MCP announcement. A community-built MCP server (`github.com/NOVA-3951/Replit-MCP`) enables Claude Desktop to directly manage Replit workspaces — creating and deleting repls, reading and writing files, managing environment variables, and triggering deployments. Replit itself added custom MCP server support to its Agent in December 2025, letting users connect hundreds of additional external tools.

**Replit AI Integrations.** Launched November 2025, this feature provides managed access to **300+ AI models** including Claude for apps users build on Replit. No separate Anthropic account needed — billing goes through Replit at transparent API rates, appearing as "Anthropic (Replit managed)" in the interface.

**Computer use.** Replit was an early adopter of Claude's computer use capability (October 2024), using it to evaluate apps visually as they're being built — Claude can perceive the UI of an application under construction and assess its quality.

## Setting up each integration path

For the **built-in Replit Agent** (the default path), setup is trivial: sign up at replit.com, choose a paid plan, and start describing what you want to build. Claude powers the experience automatically with zero configuration.

For **Claude Code in the shell**, users install the CLI tool via npm in Replit's terminal, set their `ANTHROPIC_API_KEY` in Replit's Secrets manager, and launch `claude` from the command line. Community forums note a few gotchas — Claude Code may try to run its own dev servers rather than using Replit's native compilation, requiring some configuration tweaks.

For the **community MCP server**, users add a JSON configuration block to Claude Desktop's settings pointing to the Replit MCP server, with their Replit authentication token. This enables Claude Desktop to manage repls, files, and deployments remotely. Installation is also available through Smithery: `smithery install replit-workspace --client claude`.

For the **Claude Agent SDK**, developers use Replit's official template, add their Anthropic API key to Secrets, connect services via Replit Integrations for OAuth, and build agent logic using Plan Mode. Agents can be deployed as scheduled or reserved VM deployments.

## The strategic landscape is complex and evolving

The Anthropic-Replit relationship exists within an interesting competitive dynamic. In December 2025, Google Cloud announced a **multi-year strategic partnership** with Replit, partly positioning the combination against Anthropic's own Claude Code product and Cursor. Replit is simultaneously one of Claude's largest API customers and a competitor to Claude Code in the AI coding tools market.

A head-to-head comparison published on DEV Community in 2026 captured the distinction well: Claude Code produces code that "reads like a senior developer wrote it" with excellent architectural decisions but operates only in the terminal, while Replit Agent is "the only tool that delivers end-to-end without you opening a terminal" but makes bold decisions without asking. The emerging developer consensus treats them as complementary — Claude Code for planning and architecture, Replit Agent for rapid prototyping and deployment.

The two companies clearly maintain a strong collaborative relationship despite competitive overlap, co-hosting events like the **Build India hackathon** (February 2026) with Lightspeed, and Replit continuing to upgrade to the latest Claude models as they release. A Maven course titled "Ship with Claude Code and Replit" ($1,699, next cohort April 2026) reflects the growing ecosystem around combining both tools.

## Conclusion

Claude's integration with Replit is best understood as a **multi-layered partnership** rather than a single feature. At its core, Claude is the AI brain behind Replit Agent — powering everything from code generation to autonomous debugging across a platform with **40 million+ registered users**. But the integration extends well beyond this: Claude Code runs in Replit's shell, the Agent SDK enables custom agent development, MCP bridges Claude Desktop to Replit workspaces, and managed API access makes Claude available to every app built on the platform. The relationship has driven extraordinary business results for both companies while creating multiple distinct workflows for developers — from zero-config Agent usage to advanced multi-agent SDK architectures. As of March 2026, Claude Sonnet 4.5 is the default model, and the integration continues deepening with each new model release and platform update.
