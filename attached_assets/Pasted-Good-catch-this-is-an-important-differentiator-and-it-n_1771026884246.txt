Messaging & Cryptographic Security (Clean Rewrite)

Messaging Security Powered by XMTP + MLS

DJZS uses end-to-end encrypted messaging built on Messaging Layer Security (MLS) and the XMTP network, providing modern cryptographic guarantees for both 1:1 and group communication.

Message Confidentiality

Messages are encrypted using AEAD (Authenticated Encryption with Associated Data).
Only intended participants can read the content.

⸻

Forward Secrecy

Encryption keys are ratcheted forward on every message.
If a key is compromised, past messages remain secure.

⸻

Post-Compromise Security

Group secrets are regularly rotated via MLS commit updates.
An attacker with old state cannot decrypt future messages.

⸻

Message Authentication

Messages are digitally signed, preventing impersonation and spoofing while preserving privacy.

⸻

Message Integrity

Any tampering with message content is detected cryptographically through MLS and AEAD verification.

⸻
