1) Frontend bugs to check (and fixes)

A) Price → wei mismatch on mint

Symptom: “Incorrect ETH amount” revert.
Cause: Sending 0.001 as a float instead of wei.
Fix (ethers v6):

// src/lib/mint.ts
import { ethers } from "ethers";

// Vite secret comes in as string
const priceEth = import.meta.env.VITE_SUBSCRIBE_PRICE || "0.001";
const value = ethers.parseEther(priceEth); // ✅ BigInt wei

await contract.mint({ value });

B) Network mismatch (wallet on wrong chain)

Symptom: “Contract not found” / calls fail.
**Fix: prompt to switch chain programmatically:

// src/lib/network.ts
export const BASE_MAINNET = { chainId: "0x2105", rpcUrls:["https://mainnet.base.org"] };
export const BASE_SEPOLIA = { chainId: "0x14a34", rpcUrls:["https://sepolia.base.org"] };

export async function ensureBase(chain: "mainnet"|"sepolia" = "sepolia") {
  const target = chain === "mainnet" ? BASE_MAINNET : BASE_SEPOLIA;
  // @ts-ignore
  const ethereum = window.ethereum;
  const current = await ethereum.request({ method: "eth_chainId" });
  if (current !== target.chainId) {
    try {
      await ethereum.request({ method:"wallet_switchEthereumChain", params:[{ chainId: target.chainId }] });
    } catch (e:any) {
      if (e.code === 4902) {
        await ethereum.request({ method:"wallet_addEthereumChain", params:[{ ...target, chainName: chain==="mainnet"?"Base":"Base Sepolia", nativeCurrency:{ name:"ETH", symbol:"ETH", decimals:18 } }] });
      } else { throw e; }
    }
  }
}

Call await ensureBase("sepolia") before mint.

C) ethers v5 vs v6 API mismatch

Symptom: Type errors, providers.Web3Provider not found, etc.
Fix (v6):

// src/lib/provider.ts
import { ethers } from "ethers";
// @ts-ignore
export const browserProvider = new ethers.BrowserProvider(window.ethereum);
export async function getSigner() {
  const signer = await browserProvider.getSigner();
  return signer;

const rpc = (import.meta.env.VITE_RPC_URL || "https://sepolia.base.org");

export async function hasPass(user: string) {
  const bal = await contract.balanceOf(user);
  return (bal > 0n);
}

3) Subscribe NFT “gating” helper (drop-in)

Use on the server to gate premium endpoints:

// server/gating.ts
import { ethers } from "ethers";
import ABI from "../abi/SubscribeNFT.json" assert { type:"json" };

const RPC = process.env.RPC_URL || "https://sepolia.base.org";
const ADDR = process.env.SUBSCRIBE_NFT_ADDR; // mirror Vite var in server .env

const provider = new ethers.JsonRpcProvider(RPC);
const contract = new ethers.Contract(ADDR, ABI, provider);

export async function requirePass(address: string) {
  if (!ethers.isAddress(address)) return false;
  const bal = await contract.balanceOf(address);
  return bal > 0n;
}

Use:

app.post("/api/premium", async (req,res)=>{
  const { address } = req.body || {};
  if (!(await requirePass(address))) return res.status(403).json({ error:"pass_required" });
  // continue…
});
