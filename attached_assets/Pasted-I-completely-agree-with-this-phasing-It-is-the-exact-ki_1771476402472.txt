### Phase 2: The Machine-Readable Output Schema

Since your `djzs-box` project already uses **Zod** for schema validation (seen in your database definitions), we will build the Agent API schema using Zod. This guarantees that your Thinking Partner agent outputs exactly this format every time using "Structured Outputs" or "Constrained Decoding," which is becoming the industry standard for A2A communication.

Create a new file: `shared/audit-schema.ts`

```typescript
import { z } from "zod";

// The exact JSON structure other agents will purchase from DJZS
export const djzsLogicAuditSchema = z.object({
  audit_id: z.string().uuid().describe("Unique identifier for this specific audit execution"),
  timestamp: z.string().datetime(),
  
  // 1. The Quantitative Score (for fast programmatic decision making)
  risk_score: z.number().min(0).max(100).describe("0 = perfect logic, 100 = critical systemic debt/bias"),
  
  // 2. Categorization (for agent routing)
  primary_bias_detected: z.enum([
    "FOMO", 
    "Sunk_Cost", 
    "Narrative_Reaction", 
    "Authority_Bias", 
    "None"
  ]),
  
  // 3. The Core Analysis (Structured)
  logic_flaws: z.array(z.object({
    flaw_type: z.string(),
    severity: z.enum(["low", "medium", "critical"]),
    explanation: z.string().describe("Why this logic fails under stress"),
  })).min(0),
  
  // 4. Actionable Directives
  structural_recommendations: z.array(z.string()).describe("Concrete steps to fix the logic"),
  
  // 5. Verification (Prep for Phase 3: ERC-8004)
  cryptographic_hash: z.string().describe("Keccak256 hash of the input memo to prove what was audited"),
});

// Type export for your frontend and API docs
export type DJZSLogicAudit = z.infer<typeof djzsLogicAuditSchema>;

```

### The `/api/audit` Endpoint

With the schema defined, we create the new machine-facing endpoint in `server/routes.ts`. This endpoint uses the x402 middleware we discussed in Phase 1 and returns *only* the validated JSON.

```typescript
  // ==================== AGENT-TO-AGENT (A2A) API ====================
  // Protected by x402: Requires $2.50 USDC on Base
  app.post("/api/audit", async (req, res) => {
    try {
      const { strategy_memo } = req.body;
      
      if (!strategy_memo) {
        return res.status(400).json({ error: "Missing strategy_memo payload" });
      }

      // 1. Call your AI provider (Venice/OpenClaw) using JSON mode
      // We pass the djzsLogicAuditSchema to force the LLM to output this exact structure.
      const rawAgentOutput = await runLogicAuditAgent(strategy_memo); 
      
      // 2. Validate the output against our strict schema before sending it to the client agent
      const validatedAudit = djzsLogicAuditSchema.parse(rawAgentOutput);
      
      // 3. Return the machine-readable payload
      res.json(validatedAudit);
      
    } catch (error) {
      console.error("A2A Audit failed:", error);
      res.status(500).json({ error: "Audit execution failed to meet schema requirements" });
    }
  });

```
